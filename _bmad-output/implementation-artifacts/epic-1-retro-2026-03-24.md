### 0. RLS Bugs Chỉ Detect Được Khi Test Thủ Công ⚠️ (Critical)

**Vấn đề — từ Project Lead Thắng:** Các lỗi RLS nghiêm trọng không được AI dev agent hay review agent phát hiện. Chỉ được phát hiện khi **mở app → test thực tế → check network requests**.

**Tại sao nguy hiểm:**
- RLS bugs là **silent failures** — không compile error, không runtime exception
- Query chỉ trả về empty set hoặc block thầm lặng

- AI agent review static code không thể simulate runtime JWT context

**Root cause:** CLAUDE.md đã quy định rõ `npx supabase test db` là bắt buộc, nhưng chỉ Story 1.6 thực hiện (17/17 PASS). Stories 1.1–1.5, 1.7 không có evidence chạy DB tests.

**Fix:** Enforce nghiêm ngặt — mọi story có migration mới phải chạy `npx supabase test db` và pass trước khi mark done. Không phải optional.

### Process

7. **Đây là retro đầu tiên — habit tốt cần build từ Epic 1:**
